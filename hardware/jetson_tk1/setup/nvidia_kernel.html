Install NVIDIA kernel and drivers

Requirement:

  • Host PC with Linux
  • Internet
  • USB cable
    • This is included in Jetson TK1 box.

Instruction:

  1. Login to your host PC
  2. Download NVIDIA Tegra Linux Driver Package and sample file system
    3. $ wget http://developer.download.nvidia.com/embedded/L4T/r21_Release_v3.0/Tegra124_Linux_R21.3.0_armhf.tbz2
$ wget http://developer.download.nvidia.com/embedded/L4T/r21_Release_v3.0/Tegra_Linux_Sample-Root-Filesystem_R21.3.0_armhf.tbz2
  3. extract it in host PC
    4. $ sudo tar -vxjf Tegra124_Linux_R21.3.0_armhf.tbz2
$ cd Linux_for_Tegra/rootfs
$ sudo tar jxpf ../../Tegra_Linux_Sample-Root-Filesystem_R21.3.0_armhf.tbz2
  4. Run the apply_binaries.sh script to copy the NVIDIA libraries into rootfs directory
    5. $ cd ..
$ sudo ./apply_binaries.sh
  5. Connect Jetson TK1 to host PC and boot Jetson TK1 in recovery mode
  6. Flash board
    7. This command install OS in Jetson TK1's internal storage.
    $ sudo ./flash.sh -S 14580MiB jetson-tk1 mmcblk0p1
