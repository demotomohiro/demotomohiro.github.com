Using Jetson TK1 like PC



This is simplest way to login if everything work fine.

Requirement:

  • Display with HDMI input
  • USB keyboard

Instruction:

  1. Connect USB keyboard and HDMI dispaly to Jetson tk1
  2. Connect AC adapter to Jetson tk1
    3. Jetson boot as soon as you connect AC adapter.
  3. Login as username:ubuntu/password:ubuntu
    4. Desktop don't start at the time.
